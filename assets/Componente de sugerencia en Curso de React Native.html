      Para los que no les gusta requerir sus archivo con rutas relativas aqui les dejo un plugin de babel para no usar tantos ‘…/…/…’

      Plugin: babel-plugin-module-resolver

      Mi .babelrc

       {
   "presets": ["react-native"],
   "plugins": [
     "transform-class-properties",
     ["module-resolver", {
       "alias": {
         "assets": "./assets"
       }
     }]
   ]
 }
